# PMP-Python-Music-Player-
Basic music player written in python.

A simple music player program written in **Python** that offers basic features and various themes, including minimods and background modes.

## Features
+ Make playlist with bulk choosing folder option or drag and drop song and you can do both!
+ After making a playlist you can save them in .m3u format later you can load it and listen same list again (dont change the path of the music)
+ Crossfade support
+ Sleep Timer
+ Loop and Shuffle support
+ Turkish and English support
+ Continue the last song where you left from
